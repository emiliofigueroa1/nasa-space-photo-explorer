# NASA Space Explorer

This is an frontend project using ReactJS that shows picture of space from NASA and lets the user explore these pictures.


# Resources Used:

- React ( useEffect & useState )
- JavaScript ( Json Parsing & Handling )
- CSS
- HTML 
- NASA API
